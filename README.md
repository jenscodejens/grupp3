# Julrecept

A Swedish Christmas recipe web application built by 4 members of Group 3 with React, allowing users to browse, view, and interact with traditional recipes.

## Description

Julrecept is a web platform for discovering and exploring Swedish Christmas recipes. Users can filter recipes by category, search for specific dishes, view detailed recipe information including ingredients and instructions, rate recipes, and leave comments.

## Features

- Browse a collection of Christmas recipes
- Filter recipes by category and search by title
- View detailed recipe pages with ingredients, cooking instructions, and preparation time
- Rate recipes on a 5-star scale
- Add and view user comments on recipes
- Responsive design using Bootstrap
- API documentation available via Swagger UI

## Tech Stack

- **Frontend**: React 18, React Router DOM
- **Styling**: Bootstrap 5
- **HTTP Client**: Axios
- **API Documentation**: Swagger UI React

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/jenscodejens/grupp3.git
   cd julrecept
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

The application will be available at `http://localhost:3000`.

## Usage

- Navigate to the home page to browse recipes
- Use the filter options to narrow down recipes by category or search term
- Click on a recipe card to view detailed information
- Rate recipes and add comments on the recipe detail page
- Access API documentation at `/swagger`

## API

The application communicates with a backend API for recipe and category data. Ensure the backend server is running and configured correctly.
